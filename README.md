# MVP сервиса для сбора и обработки подписей Госключом под массовыми обращениями в госорганы.

## Установка зависимостей:

`pip install -r requirements.txt`

или

`poetry install`

## Пример запуска:

`python main.py --source tests/fixtures/mass --out list.ods`

## Параметры:

`python main.py -h`
